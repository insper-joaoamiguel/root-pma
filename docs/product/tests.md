# Testes

## Casos basicos

- criar produto com `POST /products`;
- listar produtos com `GET /products`;
- consultar produto por id com `GET /products/{id}`;
- remover produto com `DELETE /products/{id}`.

## Evidencias esperadas

Os retornos devem mostrar que o cadastro e a consulta funcionam corretamente e que o produto criado pode ser reutilizado pela Order API.
